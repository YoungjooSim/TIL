1. 컴퓨터의 논리 구조('22.07.9)
- 오늘날의 컴퓨터는 60~70년 전보다 휠씬 작고 저렴하고 빠르고 안정적이다.
그래서 오늘날에 사람들은 컴퓨터를 더 많이 사용할 수 있다.

2. 프로세서 속도와 심장 박동수
1) 주 기억 장치, 보조 기억 장치, 프로세서, 디스플레이, 키보드 등 은 bus라는 여러 개의 전선으로 연결된다.
2) 프로세서 - CPU : 컴퓨터의 두뇌
core i7(패키지의 개수), 2.2Hz(1초당 22억번 펄스)
3) 주 기억 장치 = RAM : 프로세서와 컴퓨터의 다른 부분이 현재 사용 중인 정보를 저장
주기억장치는 프로세서가 현재 작업 중인 데이터뿐만 아니라 프로세서가 그 데이터로 무엇을 해야 하는지 알려 주는 명령어도 저장한다.
웹 서핑, 이메일, 페이스북을 동시에 할 수 있는 이유는 모든 일이 적합한 명령어를 메모리에 배치함으로써 가능
4) 보조 기억 장치 = 디스크 or 드라이브

3. HDD와 SDD의 차이
1) 보조 기억 장치 = 하드디스크, 하드 드라이브, SSD : 전원이 꺼져 있을 때도 정보를 유지한다.
2) SSD : 전원이 꺼져 있어도 개별 소자에 전하charge를 유지하는 회로에 정보가 전하 형태로 저장된다.
1b(바이트) : 영어 문자 한개 저장 K(1,024b) -> M(1,024k) -> G(1,024m) -> T(1,024g)

4. 가로세로 1cm 프로세서 칩
1) 컴퓨터의 전자 회로는 몇가지 기본 소자가 매우 많이 모여 만들어진다.
2) 가장 중요한 기본 소재로는 논리게이트logic gate를 들 수 있다. 논리 게이트는 한 개나 두 개의 입력 값을 바탕으로 단일 출력 값을 계산해 낸다.
또한 전압이나 전류 같은 입력 신호를 이용하여 전압이나 전류인 출력 신호를 제어한다.
3) 가장 핵심적인 부분은 트랜지스터transisor다. 트랜지스터는 기본적으로 스위치(전압의 제어를 받아 전류를 켜거나 끄는 장치) 역할을 한다.

5. 50년 넘게 유지된 무어의 법칙('22.07.10)
프로세서는 칩 하나에 프로세서 코어를 두 개 이상 배치함으로써 더 많은 트랜지스터를 활용할 수 있었다.
즉, 개별 코어의 실행 속도가 빨라진다기보다는 장착 가능한 코어의 개수가 늘면서 성능이 향상된다고 볼 수 있다.
